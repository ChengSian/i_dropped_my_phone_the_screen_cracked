Quick Start

Create

Create and connect nodes in one step by chaining their methods together. Sound flows from left to right.

//create & connect a sine, lowpass, wavershaper & compressor to output __().sine().lowpass().waveshaper().compressor().dac().play();

Configure

Set key values on audio nodes with a single argument or use JSON to set multiple parameters at once.

//create a sawtooth oscillator with a frequency of 180, a lowpass filter //with a frequency of 175 and Q of 20 and a dac node with a gain of 0.25. __().saw(180).lowpass({frequency:175,q:20}).dac(.25).play();

Connect

Connect nodes to existing nodes by referring to them using CSS style selectors. You can use type, id & class selectors

//give the lowpass an id and and the compressor a class name so we can find //them later __().square().lowpass({id:"foo"}).compressor({class:"bar"}).dac(.25); //create a reverb and connect its input to the existing square wave //and its output to the previous instantiated compressor, using //node type and class name as selectors. __("square").reverb().connect(".bar"); //create a waveshaper and connect it to the existing lowpass and //the system output using id and node name __("#foo").waveshaper().connect("dac").play();

Control

Control nodes by using selectors to get a reference and simple setters to control parameters

//create a triange wave and connect it to output //give ids to the oscillator and gain nodes __().triangle({id:"t1"}).gain({id:"g1"}).dac(); //start the oscillator __("#t1").start(); //use jquery to bind to events from the slider $("input[type='range']").bind("input",function(evt){ if(evt && evt.target) { //change the frequency of the triangle //with values from the slider __("#t1").frequency(evt.target.value); } }); //bind to the checkbox to mute the gain $("input[type='checkbox']").bind("click",function(evt){ if(evt && evt.target) { var state = evt.target.checked ? 0 : 1; //if checked, set the gain to 0 //unchecked, set gain to 1 __("#g1").volume(state); } });
